# Development-of-Sensor-Guided-Robotic-Manipulator-Arm-for-Compaction-and-Stacking-of-Domestic-Waste.-
Sensor-guided robotic manipulator for domestic waste detection, sorting, compaction, and stacking using YOLOv5, stepper-driven robotic arm control, inverse kinematics, and sensor integration. Built with Jetson Nano/Arduino for automated smart waste handling and compact storage systems.
